== Description ==

Adds a Automotive News widget for use in your WordPress sidebar. The widget displays the latest news & reviews from AllTheRides.com. 

Tested on WordPress 2.3 - WordPress 2.9.

== Installation ==

1. Download, unzip and upload the `automotive-news-sidebar-widget` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Enable the widget (`Design > Widgets`) and customise the options.

== Frequently Asked Questions ==

= How can I change the image displayed by the widget? =
Simply open the widget option page (`Design > Widgets`) and select one from the dropdown list.



= This thing is messing up my sidebar! How can I fix it? =
While the widget should work with most themes out-of-the-box, you can customize the way it is displayed easily using CSS or by simmply adding some HTML or inline styles using the widget options page (see the screenshots).
